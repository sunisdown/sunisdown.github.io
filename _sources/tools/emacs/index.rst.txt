多窗格（MULTIPLE WINDOWS）
====================================

Splitting Windowns
--------------------

C-x 2

    在当前窗口的下面再分一个窗口

C-x 3

    在当前窗口的左边分一个窗口


C-x 0

    杀死当前窗口

+------------------------------+------------------------------+------------------------------+
| key                          | command                      |                              |
+------------------------------+------------------------------+------------------------------+
| C-x 0                        | delete-windown               |     杀死当前窗口             |
+------------------------------+------------------------------+------------------------------+
| C-x 1                        | delete-other-windows         | 仅保留当窗口，杀其他窗口 |
+------------------------------+------------------------------+------------------------------+
